# My Home

This is the ArmbrustTracker project, a mobile-first application to track crossbow performance.


## Getting Started

First, run the development server:

```bash
NODE_TLS_REJECT_UNAUTHORIZED=0 npm run dev # for self-signed certificates
# or
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project is built with Next.js and uses Geist for typography.
